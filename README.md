# 🛒 Online Retail Store Dashboard

## 📌 Project Overview

The **Online Retail Store Dashboard** is a Power BI project designed to analyze retail store transaction data and provide meaningful business insights through interactive visualizations. The dashboard helps monitor sales performance, customer behavior, and overall business trends to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze retail store transaction data.
* Visualize sales performance and trends.
* Identify top-performing products and categories.
* Monitor key business metrics.
* Enable data-driven business decisions through interactive dashboards.

---

## 🚀 Getting Started

### Prerequisites

* Microsoft Power BI Desktop

### Installation

1. Clone or download this repository.
2. Open `RetailDashboard.pbix` using Power BI Desktop.
3. If prompted, reconnect the dataset `Retail-Store-Transactions.xlsx`.
4. Refresh the data if required.
5. Explore the interactive dashboard.
